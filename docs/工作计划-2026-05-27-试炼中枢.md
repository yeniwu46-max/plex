## MVP 范围

### 数据表

- `trials`：班级试炼定义（类型、星域、难度、时长、状态、奖励积分）
- `trial_participations`：学生参与/完成记录
- `system_settings`：控制中枢 JSON 配置（按班级或全局）

### API（`/api/v1`）

| 方法 | 路径 | 角色 |
|------|------|------|
| GET | `/teacher/trials?class_id=` | teacher, admin |
| POST | `/teacher/trials` | teacher, admin |
| PATCH | `/teacher/trials/:id` | teacher, admin |
| GET | `/teacher/students/:id/trials` | teacher, admin |
| GET | `/student/trials` | student |
| POST | `/student/trials/:id/join` | student |
| POST | `/student/trials/:id/complete` | student |
| GET/PUT | `/admin/settings` | admin, teacher |
| GET | `/student/learning-path` | student |
| GET | `/student/archive-insights` | student |

### 前端

- `teacherTrials.ts`、`studentTrials.ts`、`adminSettings.ts`、`studentProgress.ts`
- `TrialArenaView`、`StudentTrialView`、`TrialArenaEntry`
- `ExplorerTrialPanel`、`AdminHomeView` 配置读写
- `StarPathLabView`、`ExplorationArchivesView`、`StudentHomeView` 真实数据

### 不做

- 实时对战、题库判题、模板库持久化
