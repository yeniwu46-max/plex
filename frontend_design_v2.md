# 前端设计方案 v2.0 - 高活力&游戏化版本

## 一、设计理念升级

### 核心目标
✨ **调动学生学习积极性** + **打造沉浸式学习体验**

参考理念来源：
- **ACWing**：深蓝+渐变 + 难度分级视觉化
- **洛谷**：活力绿 + 勋章系统 + 暗黑主题选项
- **力扣**：橙色热情 + 成就解锁 + 实时反馈

**创新点**：融合编程平台的激励机制，打造学习游戏化系统

## 三、游戏化激励系统设计

### 1. **学生等级系统**

```
Lv 1 新手探索者 (0-99 积分)      🟢
Lv 2 知识见习生 (100-299 积分)    🟢
Lv 3 学习骑士 (300-699 积分)      🟢
Lv 4 大师学者 (700-1499 积分)     🟡
Lv 5 知识守护者 (1500+ 积分)      🟠

视觉表现：
- 等级徽章带发光效果
- 等级提升时有升级动画（烟火特效）
- 名片上显示等级和称号
```

### 2. **勋章/徽章系统**

| 勋章 | 获取条件 | 视觉 | 稀有度 |
|------|---------|------|--------|
| 🎯 **初心者** | 完成第一个课程 | 蓝色徽章 | ⭐ 普通 |
| 🔥 **连续学习者** | 连续学习7天 | 红色徽章+火焰边框 | ⭐⭐ 稀有 |
| 🏆 **全能大师** | 完成所有课程 | 金色徽章+王冠 | ⭐⭐⭐ 稀有 |
| ⚡ **闪电快手** | 单周完成5个任务 | 黄色徽章+闪电 | ⭐⭐ 稀有 |
| 🌟 **完美主义者** | 获得10个满分评价 | 彩虹徽章+星光 | ⭐⭐⭐⭐ 极稀有 |
| 🎓 **优秀学员** | 平均分>90 | 红色学位帽徽章 | ⭐⭐⭐ 稀有 |

**视觉效果**：
- 未解锁勋章显示灰色+问号
- 解锁时有"彩带飘落"动画
- 点击勋章可查看获取条件

### 3. **排名系统**

```
🏅 班级排名榜（每周更新）
  
位置    学生名    积分    变化
🥇 1    张三      2850   ↑3
🥈 2    李四      2650   ↓1  
🥉 3    王五      2400   ↑1
  4    赵六      1950   →
  5    孙七      1800   ↓2

设计要点：
- 排名改变时有上升/下降箭头
- Top 3 额外显示奖杯颜色
- 可以看到上周排名对比
```

### 4. **成就解锁动画**

当学生完成任务时：
```
触发事件 → 成就解锁弹窗
          ↓
       金色烟火 🎆
          ↓
      "恭喜解锁"
          ↓
    勋章旋转特效 ✨
          ↓
    积分飘起来 +50
```

## 六、核心微交互设计

### 1. 页面加载动画
```
加载中: 脉冲绿色圆点 ● ● ●
```

### 2. 按钮hover效果
```css
正常: 活力绿背景 + 白字
hover: 颜色加深 + 轻微上浮 (2px) + 阴影
active: 颜色更深 + 按下感
```

### 3. 成就解锁弹窗
```
触发条件 → 页面中心出现金色光圈
        → 勋章从中心放大(200% → 100%)
        → 彩带从上往下飘落
        → 积分数字从下往上飘起
        → "恭喜解锁XXX勋章"文字淡出
```

### 4. 进度条动画
```
静止: 固定百分比宽度
完成时: 进度条变绿 + 右边界出现✓勾号
```

## 七、响应式布局（优先PC）

### PC端（1200px+）- 完整版
- 三栏布局：导航 + 内容 + 侧边栏

- 一栏布局：顶部菜单，内容全屏
- 卡片竖排堆积

## 九、动画库选择

使用 **Animate.css** 或 **AOS (Animate On Scroll)**

关键动画：
- fadeIn: 页面元素进入
- slideUp: 卡片滑上
- pulse: 限时任务脉冲
- bounceIn: 成就解锁
- zoomIn: 勋章出现
